## WebAR-Spaceship 

### Description 
WebAR project for rendering a spaceship ( built in Blender ) in the real world.

### Libraries/Frameworks used 
1. HTML 
2. CSS 
3. Javascript 
4. Three.js
5. WebGL
6. Blender ( exporting .gltf file ) 
7. ngrok
8. WebXR
